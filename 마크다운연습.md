## CLI(command line inferface)
--------------
>CLI는 커맨드라인 인터페이스를 뜻한다.터미널로 상호작용

### 터미널 명령어정리

명령어|설명
|------|------|
mkdir| 폴더생성
touch|파일생성
ls|현쟈폴더의 목록출력
cd|다른폴더로 이동
rm|파일삭제 -r폴더삭제

예시
```
$mkdir happy
```
